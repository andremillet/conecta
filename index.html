Conecta

Instale o Conecta executando o seguinte comando no seu terminal:

curl -sSL https://andremillet.github.io/conecta/install.sh | bash

O que é o Conecta?

O conecta é uma ferramenta de linha de comando que permite baixar e executar scripts Python diretamente de um repositório no GitHub, de forma rápida e sem a necessidade de clonar o projeto inteiro.

Basta fornecer o caminho para o script e o conecta cuida do resto.

Exemplo de Uso

Para executar um script, utilize o caminho no formato usuario/repositorio/branch/caminho/para/script (sem a extensão .py). 

conecta andremillet/prognosys/conduta-extractor/tui
